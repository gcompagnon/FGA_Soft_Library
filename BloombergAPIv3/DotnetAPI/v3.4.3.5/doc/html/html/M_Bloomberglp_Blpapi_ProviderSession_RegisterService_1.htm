RegisterService Method (serviceName, providerIdentity, registrationOptions)
Namespaces ► Bloomberglp.Blpapi ► ProviderSession ► RegisterService(String, Identity, ServiceRegistrationOptions)

Attempt to register the service identified by the specified 'uri' and block until the service is either registered successfully or has failed to be registered. The specified 'providerIdentity' is used to verify permissions to provide the service being registered. Returns true if the service is registered successfully and false if the service cannot be registered successfully.

The 'uri' must begin with a full qualified service name. That is it must begin with "//{namespace}/{service-name}[/]". Any portion of the 'uri' after the service name is ignored.

Before registerService() returns a SERVICE_STATUS Event is generated. If this is an asynchronous ProviderSession then this Event may be processed by the registered Event before registerService() has returned.
