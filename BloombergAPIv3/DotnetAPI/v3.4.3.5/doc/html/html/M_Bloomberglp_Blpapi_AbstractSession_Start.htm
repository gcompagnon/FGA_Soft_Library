Remarks
Issues a blocking call to start the session. This call returns true if the Session was started successfully or falseotherwise. Also a Event.EventType.SESSION_STATUS event is published containing a message with the result of the request. If the Session was started successfully a "SessionStarted" message will be generated. If the start failed a "SessionStartupFailure" message will be generated.

Note: If this is an asynchronous Session then the Event.EventType.SESSION_STATUS may be processed by the registered EventHandler before this method has returned.

A Session may only be started once.
