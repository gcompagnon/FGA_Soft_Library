Parameters
uri (String)
URI of the service that is to be opened.
Return Value
A handle to the service if it is already open; null otherwise.
Remarks
The uri must contain a fully qualified service name. That is, it must be of the form "//[namespace]/[service-name]". If the specified service is not already open, a null handle is returned.
