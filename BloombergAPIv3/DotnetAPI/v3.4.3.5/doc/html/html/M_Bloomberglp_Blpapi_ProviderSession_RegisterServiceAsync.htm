Return Value
The actual correlationId that will identify Events generated as a result of this call.
Remarks
The specified 'providerIdentity' is used to verify permissions to provide the service being registered.

The 'uri' must begin with a full qualified service name. That is it must begin with "//{namespace}/{service-name}[/]". Any portion of the 'uri' after the service name is ignored.

The application must monitor events for a {@link EventType#SERVICE_STATUS} event which will be generated once the service has been successfully registered or registration has failed.
